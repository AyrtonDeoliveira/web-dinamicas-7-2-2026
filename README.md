# Diseño de Aplicaciones Web Dinámicas
### 7mo 2da — Programación

Repositorio de ejercicios y actividades de la materia. Cada carpeta corresponde a una unidad temática con sus respectivos archivos de la actividad.

---

## Contenido actual

### 📂 Vectores
Ejercicios de declaración, recorrido y manipulación de vectores con JavaScript.

### 📂 Arrays
Ejercicios con arrays: creación, métodos básicos y operaciones de búsqueda y modificación.

### 📂 Arrays de objetos
Trabajo con colecciones de objetos: acceso a propiedades, recorrido y renderizado dinámico en el DOM.

### 📂 Bases de datos
Introducción al manejo y consulta de datos estructurados.

### 📂 JSON
Ejercicios de carga y visualización de datos desde archivos `.json` externos usando `fetch`. Cada ejercicio tiene sus archivos separados: `index.html`, `style.css`, `app.js` y el archivo `.json` correspondiente.

---

## Cómo usar los ejercicios

Los ejercicios que cargan archivos `.json` con `fetch` necesitan ejecutarse desde un servidor local (no con doble clic). Opciones:

- **VS Code**: instalar la extensión **Live Server** y click en *Go Live*
- **Terminal**: entrar a la carpeta del ejercicio y ejecutar:
  ```bash
  python3 -m http.server
  ```
  Luego abrir `http://localhost:8000` en el navegador.
